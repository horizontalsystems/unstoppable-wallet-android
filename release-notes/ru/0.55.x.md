## 🚀 Обновление версии 0.55.0
_Дата релиза: 14 мая 2026_

### ✨ Улучшения

- **Добавлена поддержка аппаратного кошелька Trezor**
  Поддерживаются все модели для `BTC`, `LTC`, `BCH`, `DOGE`, `ETH`, `BSC`, `Polygon`, `Arbitrum`, `Optimism`, `Base` и `Stellar`.
  Для `Dash` поддерживаются только `Trezor One` и `Model T`.
  Для `Solana` поддерживаются только `Model T`, `Safe 3`, `Safe 5` и `Safe 7`.

- **Добавлена новая премиум-функция «Псевдо-калькулятор»**
  Приложение может выглядеть и работать как обычный калькулятор.
  Если результат вычисления совпадает с PIN-кодом, открывается вход в кошелёк.

- **Доработан фильтр уведомлений о новых транзакциях**
  В push-уведомлениях больше не отображается спам.

- **Приложение больше не добавляет неизвестные токены автоматически**
  Если приходит транзакция с неизвестным токеном, такой актив больше не появляется в кошельке автоматически.
  Это позволяет скрывать нежелательные активы.

- **Переработана логика отображения невыплаченного остатка для PirateCash и Cosanta**

- **Переработана механика восстановления по seed-фразе не на английском языке**
